Beranda

Sukabumi Game Center merupakan pusat perbelanjaan game terbesar se-sukabumi
yang berletak di kecamatan Sukaraja Kabupaten Sukabumi. Disini menyediakan
berbagai kebutuhan gaming mulai dari berbagai merk konsol dari beberapa
generasi, berbagai game dari game generasi terdahulu sampai generasi terbaru.
Ditempat ini juga menyediakan berbagai accecoris untuk kebutuhan gaming,
gunak memberikan pengalaman bermain game yang memuaskan

Dari beberapa judul game yang ternama kami juga menyedikan di bagian list game
untuk bisa anda eksplor lebih dalam dari segi status ketersediaan barang
hingga harga.

Note : untuk pertanyaan dan masukkan bisa anda kirim melalui kotak saran
